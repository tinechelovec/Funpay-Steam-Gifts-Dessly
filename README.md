# Funpay Steam Gifts (Dessly)
🚀 Бот для автоматического продажи подарков Steam через Dessly на FunPay  
📌 Сейчас в стадии бета-тестирования
      
## Что из себя представляет бот?  
Это Python-скрипт, который:  
✔ Коммиссия -15% на отправку подарков.   
✔ Настраивается все в одном файле.   
✔ Автоматически понимает, что за подарок.  
✔ Пополнение от 100$.  
  
## Что нужно для работы бота?  
1. Зайти на сайт [DesslyHUB](https://desslyhub.com/), написать менеджеру и у него попросить данные.
2. Установка Python, бота  и библиотек
```pip install -r requirements.txt```
3. Настройка .env
```
FUNPAY_AUTH_TOKEN=golden_key
DESSLY_API_KEY=DESSLY_API_KEY
AUTO_REFUND=true/false
```
